- Python 2.7

# Installation Tools Segurity
- sudo su
- git clone https://github.com/ViKtorMacias/UbuntuForWork.git && cp UbuntuForWork/ToolsSegurity.py /usr/bin/ToolsSegurity
- chmod +x /usr/bin/ToolsSegurity
